Se migro todo el codigo css a SAAS. 


Se creo la carpeta base, donde se encuentra codigo base como:


Listas: Para los colores.

Extend: Para el codigo que se usa en los input de todos los formularios.

Maps: Para el estilo de un boton.

Variables: Para definir medidas estandar de tipografias, margenes, colores.




Se creo la carpeta layout, donde se encuentra el codigo de elementos que se repiten en todos los html (mixin) como el header, footer,
el formulario de contacto, el layout de los html hechos con grid. (La pagina de contacto esta maquetada con Bootstrap 5).


Se creo la carpeta Views, donde se encuentra el codigo de cada uno de los html junto con el codigo de las medias queries.
