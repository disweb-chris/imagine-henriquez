El siguiente proyecto se realizo con el objetivo de aprobar el curso Desarrollo Web de CoderHouse.

El cliente es una agencia de marketing digital y desarrollo web que esta emprendiendo. Su marca se caracteriza por tener tematica espacial,
por lo cual se usaron elementos graficos orientados al universo, planetas y astronautas como puntos visuales en todo el proyecto para acompañar
la identidad de la marca. Se busco hacer una web corta y concisa donde el protagonista sea la parte visual (imagenes), para animar al 
usuario a comunicarse.

En las paginas de 

• Inicio 
• Desarrollo Web
• Marketing Digital
• Diseño

Se uso para el layout Grid, Flexbox y Box Modeling.

En la pagina de Contacto se uso para el layout Bootstrap 5. 

El menu hamburguesa que se ve en toda la web tambien fue realizado con Bootstrap 5 y adaptado al estilo visual de la web.

-------------- ........... --------------

Se utilizo SAAS como pre procesador de CSS.

Se creo la carpeta base, donde se encuentra codigo base como:


• Listas: Para los colores.

• Extend: Para el codigo que se usa en los input de todos los formularios.

• Maps: Para el estilo de un boton.

• Variables: Para definir medidas estandar de tipografias, margenes, colores.


Se creo la carpeta layout, donde se encuentra el codigo de elementos que se repiten en todos los html (mixin) como el header, footer,
el formulario de contacto, el layout de los html hechos con grid (menos la de contacto).


Se creo la carpeta Views, donde se encuentra el codigo de cada uno de los html junto con el codigo de las medias queries.
